# 📌 Todoist — Pin

Uma extensão para Chrome que permite **fixar tarefas do Todoist** na tela, mantendo acesso rápido e fácil às suas prioridades, sem perder de vista a lista principal.

## ✨ O que é?

**Todoist — Pin** é uma extensão minimalista que adiciona um botão discreto ao modal de tarefas do Todoist, permitindo:

- 📌 Fixar tarefas importantes em um widget flutuante
- 💾 Armazenar tarefas fixadas localmente (persistente)
- 🚀 Acesso rápido com um clique
- 🔄 Retornar ao contexto original após abrir uma tarefa fixada
- 🎯 Lista limpa e organizada de tarefas prioritárias
- ⚡ Zero impacto no desempenho do Todoist

---

## 🎯 Como funciona?

### 1️⃣ Instale a extensão
Baixe na [Chrome Web Store](#) ou carregue localmente em modo desenvolvimento.

### 2️⃣ Abra uma tarefa no Todoist
Clique em qualquer tarefa para abrir o modal de detalhes.

### 3️⃣ Clique no ícone
O botão aparece ao lado do botão fechar, na barra superior do modal.

### 4️⃣ Acesse suas tarefas fixadas
Use o widget flutuante no canto inferior direito para abrir qualquer tarefa pinada instantaneamente.

## 💻 Instalação

### Para Desenvolvimento

```bash
# Clone o repositório
git clone https://github.com/tihhgoncalves/todoist-pin.git
cd todoist-pin

# Instale as dependências
npm install

# Gere o build
npm run build
```
